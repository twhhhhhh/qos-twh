第 2 頁：列表的運算

初始列表設定為：A = [20, 18, 40, 30, 10, 5]

A. 計算平均值

展開檢視 Python 程式碼及解釋 
A = [20, 18, 40, 30, 10, 5]
total_sum = 0

# i 將由 0 跑到 5 (不包含 6)
for i in range(0, 6):
    total_sum = total_sum + A[i]

average = total_sum / 6
print("平均值:", average)

解釋： 迴圈會依序透過索引值 i 讀取列表內的元素。我們建立一個變量 total_sum 來累加每次讀取到的值。迴圈結束後，再除以元素總數 6 來求得平均。

B. 找出最大值與最小值

展開檢視 Python 程式碼及解釋 
A = [20, 18, 40, 30, 10, 5]
max_val = A[0] # 先假設第一個是最大的
min_val = A[0] # 先假設第一個是最小的

for i in range(0, 6):
    if A[i] > max_val:
        max_val = A[i]
    if A[i] < min_val:
        min_val = A[i]

print("最大:", max_val, "最小:", min_val)

解釋： 一開始先將列表的第一個元素 (索引 0) 預設為最大與最小值。隨後使用迴圈檢查每一個元素，若發現更大的值就更新 max_val，發現更小的值就更新 min_val。

C. 找出雙數項目的數量

展開檢視 Python 程式碼及解釋 
A = [20, 18, 40, 30, 10, 5]
even_count = 0

for i in range(0, 6):
    if A[i] % 2 == 0:
        even_count = even_count + 1

print("雙數數量:", even_count)

解釋： 使用 % 2 == 0 來判斷數字除以 2 的餘數是否為 0 (即判斷是否為雙數/偶數)。如果成立，計數器 even_count 就會加 1。

第 3 頁：列表的元素操作

初始列表設定為：A = [20, 18, 40, 30, 10, 5]

A. 將數值 28 插入到索引 3

觀察底層運算時，後方的元素需要先向右「順延」騰出空間，才能將新數值放入指定位置。

展開檢視 Python 程式碼及解釋 
A = [20, 18, 40, 30, 10, 5]

# Python 內建方法：
A.insert(3, 28) 
print(A)

# (背後原理概念模擬)：
# A.append(0)  先擴展空間
# for i in range(5, 2, -1):
#     A[i] = A[i-1]   將元素往右移
# A[3] = 28

解釋： 在 Python 中，我們只需使用 .insert(index, value) 即可完成。但對電腦記憶體而言，在中間插入資料時，從該位置到結尾的所有元素都必須往後移動一格，才能騰出空位存放新資料。

B. 移除索引 2 的內容並騰空最後位置

觀察底層運算時，移除中間的元素後，後方的元素會向左「填補」空缺。

展開檢視 Python 程式碼及解釋 
A = [20, 18, 40, 30, 10, 5]

# Python 內建方法：
A.pop(2) # 或 del A[2]
print(A)

# (背後原理概念模擬)：
# for i in range(2, 5):
#     A[i] = A[i+1]   將元素往左移填補
# A.pop()             刪除最後多餘的空間

解釋： Python 中可以使用 A.pop(2) 或 del A[2] 刪除指定索引的元素。底層操作上，這會導致該索引後面的所有元素往前移動一格覆蓋掉目標值，並縮短陣列的長度。

第 4 頁：列表的進階操作

初始列表設定為：A = [5, 10, 18, 20, 30, 40]

A. 檢查列表是否由小至大排序

展開檢視 Python 程式碼及解釋 
A = [5, 10, 18, 20, 30, 40]
is_sorted = True

for i in range(0, 5): # 注意範圍只需到長度-1
    if A[i] > A[i+1]: # 如果前一個比後一個大
        is_sorted = False
        break # 已發現未排序，提早結束迴圈

if is_sorted:
    print("TRUE")
else:
    print("FALSE")

解釋： 要確認是否由小到大排序，我們只需比較相鄰的兩個元素（A[i] 和 A[i+1]）。如果發現任何一個左邊大於右邊的狀況，就代表未排序，可以直接判定為 FALSE 並結束檢查。

B. 將列表內容倒轉 (Reverse)

透過交換首尾兩端的元素 (前後對調) 來實現倒轉。

展開檢視 Python 程式碼及解釋 
A = [5, 10, 18, 20, 30, 40]

# Python 內建方法：
A.reverse()
print(A)

# (自定邏輯：首尾交換法)
# length = len(A)
# for i in range(0, length // 2):
#     temp = A[i]
#     A[i] = A[length - 1 - i]
#     A[length - 1 - i] = temp

解釋： Python 提供了方便的 .reverse() 方法。若要自己編寫邏輯，常見做法是將索引 0 與最後一個交換，索引 1 與倒數第二個交換...直到中間位置，這樣迴圈次數只要列表長度的一半。
